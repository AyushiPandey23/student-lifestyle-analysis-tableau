# 📊 Student Lifestyle Analysis Dashboard (Tableau + SQL)

## Problem Statement

This project focuses on analyzing student lifestyle data to understand factors affecting academic performance and overall well-being.  
The dashboard helps in identifying patterns related to academic pressure, financial stress, study satisfaction, sleep duration, and study habits.

It provides a comprehensive view of how different factors influence student behavior and daily routines.

---

## Steps Followed

- Step 1 : Loaded dataset into SQL Server for initial data handling  
- Step 2 : Performed basic queries to explore and validate the data  

- Step 3 : Imported data into Tableau  
- Step 4 : Performed data cleaning and preparation  

- Step 5 : Created calculated fields:
  - Student Count  
  - Average Study Hours  

- Step 6 : Built separate analytical views for:
  - Academic Pressure  
  - Financial Stress  
  - Study Satisfaction  
  - Sleep Duration  
  - Study Hours  

- Step 7 : Designed KPIs:
  - Total Students  
  - Average Study Hours  

- Step 8 : Developed a final interactive dashboard combining all views  

- Step 9 : Applied filters and enabled interactivity across all visuals  

- Step 10 : Enabled cross-filtering where charts dynamically update based on user interaction

## Insights

### [1] Study Behavior
Study hours show a generally increasing trend, indicating varying levels of academic engagement among students.

### [2] Sleep Patterns
Sleep duration varies across students, showing no single dominant pattern.

### [3] Academic Pressure
Academic pressure levels are moderately distributed, suggesting most students experience balanced pressure.

### [4] Financial Stress
Financial stress is present across different levels, indicating its impact on student lifestyle.

### [5] Interactive Analysis
Cross-filtering and interactivity allow users to explore relationships between different lifestyle factors effectively.
